Outras formatações

Codigo fonte / Pré formatação

Utilizando a tag code, podemos colocar o texto com espaçamento simetricos, otimo pra ler codigos.

Utilizando a tag pre, pode deixar o codigo com a mesma formatação de quando se está escrevendo-o

Citações

Utilizando a tag q, podemos fazer um texto em formato de citação.. Pra tudo nessa vida tem um jeito, só não tem jeito pra morte..
Também podemos utilizar a tag blockquote para citações, dessa forma deslocando-as um pouco..

Ande com Pedro, ria com Pedro, coma com Pedro.. mas sempre de olho em Pedro.
ao meu ver bem mais bonito.

Abreviações

Utilizando a tag abbr podemos explicar uma abreviação, como exempre INSS.
colocando o mouse em cima da pra ver o nome completo.

Texto invertido

Utilizando a tag bdo r, podemos criar textos com escrita invertida, dando como exemplo,
Essa beleza aqui.
