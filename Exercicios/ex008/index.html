Formatação de textos

Principais Formatações

Negrito / Destaque

Utilizando a tag b podemos criar um termo em Negrito. (Não semantica)

Porém a forma certa, sendo assim, de forma semantica, é utilizando a tag STRONG (Semantica)

Italico / Ênfase

Utilizando a tag i podemos criar um termo em Italico. (Não Semantica)

Porém a forma certa, sendo assim, de forma semantica, é utilizando a tag em (Semantica)

Texto marcado / Marcação

Utilizando a tag mark, podemos criar um texto marcado.

Texto grande e pequeno

Utliziando a tag big, podemos criar um texto grande, meio obsoleto, mas ainda funciona, e utilizando a tag small podemos criar um texto pequeno.

Texto riscado/excluido

Utilizando a tag del, podemos colocar um texto riscado.

Texto sublinhado

Utilizando a tag ins, podemos colocar um texto sublinhado, também possivel na tag u, porém não semanticamente.

Texto Superior/Sobrescrito

Utlizando a tag sup, podemos criar um texto sobrescrito, fica elevado na frase, como exemplo, utlizado em x20+7.

Texto Inferior/Subscrito

Utlizando a tag sub, podemos criar um texto subscrito, fica abaixo do nivel da frase, como exemplo, utilizado em H20.
